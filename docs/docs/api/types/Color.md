# Color

RGB color representation. Used for drawing and visualization purposes.

```typescript
type Color = {
  r: number;
  g: number;
  b: number;
};
```

## Properties

| Property | Type     | Description             |
| -------- | -------- | ----------------------- |
| `r`      | `number` | Red component (0-255)   |
| `g`      | `number` | Green component (0-255) |
| `b`      | `number` | Blue component (0-255)  |
