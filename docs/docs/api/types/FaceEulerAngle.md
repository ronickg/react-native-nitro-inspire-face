# FaceEulerAngle

3D orientation angles of a detected face.

```typescript
type FaceEulerAngle = {
  roll: number;
  yaw: number;
  pitch: number;
};
```

## Properties

### roll

- Rotation around the Z-axis (face tilting left/right).

### yaw

- Rotation around the Y-axis (face turning left/right).

### pitch

- Rotation around the X-axis (face looking up/down).
