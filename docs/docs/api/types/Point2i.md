# Point2i

2D point with integer coordinates. Used for pixel-precise positioning.

```typescript
type Point2i = {
  x: number;
  y: number;
};
```

## Properties

| Property | Type     | Description                |
| -------- | -------- | -------------------------- |
| `x`      | `number` | X-coordinate as an integer |
| `y`      | `number` | Y-coordinate as an integer |
